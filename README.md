<h2 align="center">DZSA Launcher – Server Listing Keeper</h2>

---

## About

A utility script for [DZSA Launcher](https://dayzsalauncher.com/) that ensures your DayZ server stays listed. DZSA has a habit of delisting servers at random — this script automatically re-registers your server to keep it visible to players.

## Configuration

Before running, open the script and set your server details:

```bat
SET SERVERIP=YOURSERVERIP
SET QUERYPORT=YOURSERVERPORT
```

| Variable | Description |
|---|---|
| `SERVERIP` | Your server's public IP address |
| `QUERYPORT` | Your server's query port |
